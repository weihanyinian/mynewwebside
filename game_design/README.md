## 🎯 设计概要

### 游戏类型
- **品类**：Roguelike + 割草 + 动作RPG
- **目标**：爽感优先，每30秒感受到变强
- **平台**：PC + 移动端双端适配

### 核心数据

| 项目 | 内容 |
|------|------|
| 角色数 | 3个（魔法少女/剑圣/召唤师） |
| 技能数 | 60个（5大类×12个） |
| 技能升级 | 每技能3级（消耗技能点） |
| 等级上限 | 50级 |
| 关卡 | 15波次 + 3个Boss |
| 协同效果 | 8大Build组合 |

### 技能分类

```
🔥 攻击技能（A01-A12）：12个主动伤害技能
🛡️ 防御生存（B01-B12）：12个生存/护盾技能
✨ 魔法增幅（C01-C12）：12个Buff/增伤技能
🌀 召唤辅助（D01-D12）：12个召唤物技能
🎲 肉鸽特殊（E01-E12）：12个稀有橙装级词条
```

## 🎮 技术方案

### 前端
```
Vue 3 + Pinia + Phaser 3
├─ 游戏渲染：Phaser 3（嵌入Vue组件）
├─ 状态管理：Pinia
├─ UI风格：玻璃态（与现有项目一致）
└─ 适配：CSS Container Queries
```

### 后端
```
Spring Boot 3.x + MyBatis Plus + JWT
├─ 认证：JWT Token
├─ 数据库：MySQL
├─ 存档：增量同步
└─ 安全：游戏逻辑全在前端（防作弊）
```
